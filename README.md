# Fsidecar Beta

Public beta releases for Fsidecar — a local-first memory sidecar for LLM workflows.

Fsidecar helps preserve project context so you don’t have to keep re-explaining your work to AI assistants.

---

## 🚧 Status

Early Windows desktop beta.

---

## ⬇️ Download

👉 Check Releases for the latest build.

---

## 🧠 What it does

- Captures workflow context
- Stores structured memory locally
- Builds reusable prompt injections
- Runs as a local daemon + UI

---

## ⚠️ Notes

- Local-first (no cloud dependency)
- Early stage — expect rough edges
- No telemetry
