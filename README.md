# Fsidecar Beta

Public beta releases for Fsidecar — a local-first memory sidecar for LLM workflows.

Fsidecar helps preserve project context so you do not have to keep re-explaining your work to AI assistants.

## Status

This is an early Windows-first desktop beta.

## Download

See Releases for the latest packaged build.

## What it does

- Captures workflow context
- Stores structured memory locally
- Builds reusable prompt injections
- Runs as a local daemon with a desktop UI

## Notes

- Local-first
- Early stage
- Expect rough edges
